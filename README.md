# Ping Monitor (Локальный сетевой монитор доступности)

[![Build and Release](https://github.com/Temofeika/ping/actions/workflows/build.yml/badge.svg)](https://github.com/Temofeika/ping/actions/workflows/build.yml)

**Ping Monitor** — это автономная консольная программа для Windows (`.exe`), предназначенная для непрерывного отслеживания доступности устройств в локальной сети, записи истории в встраиваемую базу данных **SQLite** и анализа обрывов связи за любые заданные интервалы времени.

## Особенности проекта

1. **Работа в Windows без прав администратора**: Программа использует нативный Win32 API (`iphlpapi.dll` -> `IcmpSendEcho`), что позволяет запускать `.exe` файл под обычной учетной записью пользователя.
2. **Два режима работы**:
   - **Интерактивный режим**: Просто запустите `ping-monitor.exe` двойным кликом в проводнике. Откроется текстовое меню, где можно выбрать устройство и проверить историю обрывов связи.
   - **Режим командной строки (CLI)**: Поддержка субкоманд и флагов для скриптов и автоматизации.
3. **Локальное хранение (SQLite без CGO)**: Все логи сохраняются в компактный файл `ping_history.db` с мгновенным поиском по индексам.
4. **Умный анализ обрывов связи**: Алгоритм автоматически объединяет серии последовательных потерянных пакетов в события сбоя с указанием точного времени начала, окончания и длительности простоя.
5. **Сборка через GitHub Actions**: Полный CI/CD пайплайн для автоматической сборки бинарных файлов для Windows, Linux и macOS.

## Как запустить

### 1. Быстрый запуск (Интерактивное меню)

Дважды кликните по файлу `ping-monitor.exe` в проводнике Windows или запустите его в терминале без аргументов:

```powershell
.\ping-monitor.exe
```

Откроется меню:
```text
============================================================
         МОНИТОРИНГ НЕПРЕРЫВНОСТИ ПИНГА В ЛОКАЛЬНОЙ СЕТИ
============================================================
 Выберите действие:
   1. Запустить непрерывный мониторинг пинга до устройства
   2. Проверить историю обрывов связи за промежуток времени
   3. Показать общую статистику по всем сохраненным устройствам
   0. Выход из программы
```

---

### 2. Запуск через командную строку (CLI)

#### Непрерывный мониторинг
Запуск мониторинга IP-адреса локального роутера с интервалом в 1 секунду:
```powershell
.\ping-monitor.exe monitor --target 192.168.1.1 --interval 1 --db ping_history.db
```

#### Проверка истории обрывов связи за период
Проверка доступности за последние 24 часа:
```powershell
.\ping-monitor.exe check --target 192.168.1.1 --from "24h" --to "now"
```

Проверка за точный промежуток времени:
```powershell
.\ping-monitor.exe check --target 192.168.1.1 --from "2026-07-08 10:00:00" --to "2026-07-08 12:00:00"
```

#### Пример отчета программы
```text
======================================================================

======================================================================

 Период анализа : с 2026-07-08 10:00:00 по 2026-07-08 12:00:00
----------------------------------------------------------------------
 Отправлено запросов : 7200
 Успешных ответов    : 7185
 Потеряно пакетов    : 15
 Доступность (Uptime): 99.79%
 Среднее время (RTT) : 1.4ms (мин: 500µs, макс: 8.2ms)
======================================================================

[ВНИМАНИЕ] Обнаружено сбоев связи: 1! Общее время простоя: 15s

Начало сбоя          | Конец сбоя           | Длительность | Пакеты   | Причина
---------------------------------------------------------------------------
2026-07-08 10:45:12  | 2026-07-08 10:45:27  | 15s          | 15       | таймаут ответа (код ICMP: 11010)
```

## Сборка из исходного кода

Проект написан на Go 1.22+. Для ручной сборки выполните:

```powershell
# Сборка исполняемого файла для Windows
go build -ldflags="-s -w" -o ping-monitor.exe .

# Запуск тестов
go test -v ./...
```

Автоматическая сборка `.exe` файла также настроена через **GitHub Actions** в файле `.github/workflows/build.yml`. При каждом коммите и создании релиза готовые скомпилированные бинарники загружаются в раздел Artifacts и Releases репозитория [https://github.com/Temofeika/ping](https://github.com/Temofeika/ping).
